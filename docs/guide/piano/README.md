### 踏板
延音踏板(最右侧)作用：让声音保持更长时间，让声音延续。

原理：踩下延音踏板会使所有的制音器都抬起，使声音延续。

踩踏板时机：音符发声后立即踩下踏板，松开踏板以消音

换踏板原因：弹奏不同区间音的琴键，会是声音杂乱

切换时机：每次变换和弦都要换踏板

### 5. 传统钢琴与电钢琴的区别
钢琴是物理敲击发声的原声乐器
电钢琴是电子采样发声的数码设备

发声原理：
· 钢琴：按下琴键，联动琴槌敲击琴弦，通过巨大共鸣箱扩音，发出真实物理震动的声音。
· 电钢琴：琴键触发预先录好的顶级钢琴声音采样，通过芯片计算后，由喇叭播放

音色与音量：
· 钢琴：音色会随力度、踏板的组合产生无穷变化，声音临场感无与伦比。缺点是音量不可调，练习时可能扰民，且每年需要定期调律1-2次。重200斤以上。
· 电钢琴：音色本质上是录音，虽能忠实还原顶级钢琴，但动态和变化不如真钢丰富。优点是可插耳机静音练习，也无需调律

总结：
电钢琴：本质是钢琴录音播放，可插耳机不扰民，便携，主要是便宜（¥200-1000）。
传统机械钢琴：重达200斤以上，音量不可调，容易扰民，需要定期调律保养。但音色很好，琴键手感也好。价格（1w-3w，二手¥3500）。
